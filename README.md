# 📘 Kayy Reviser

## Description

Kayy Reviser est une plateforme de révision destinée aux élèves de Terminale au Sénégal.

## Fondateur

Mouhamed Ndiaye

## Objectif

Aider les élèves à réussir leur baccalauréat grâce à une plateforme moderne, interactive et accessible.

## Technologies

- HTML5
- CSS3
- JavaScript
