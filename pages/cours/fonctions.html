📈 Les Fonctions

Comprendre la notion de fonction et savoir résoudre les exercices du Bac.

📖 Résumé du cours

Une fonction est une application qui associe à chaque nombre \(x\) un unique nombre noté \(f(x)\).


Exemple : \[ f(x)=2x+3 \]


Calculer : \[ f(2)=2\times2+3=7 \]

📚 Définitions importantes

📌 Image d'un nombre

L'image d'un nombre x est la valeur f(x) obtenue après le calcul.


Exemple :

\[ f(x)=2x+1 \] \[ f(3)=7 \]

📌 Antécédent

Un antécédent de k est un nombre x tel que :

\[ f(x)=k \]

Exemple :

\[ 3x-2=7 \] \[ x=3 \]

📌 Domaine de définition

C'est l'ensemble des valeurs de x pour lesquelles la fonction existe.


Exemple :

\[ f(x)=\frac{1}{x-2} \]

La fonction n'est pas définie pour x = 2.

💡 Points essentiels

📌 Domaine de définition

Déterminer les valeurs possibles de \(x\).

📈 Image

Calculer \(f(x)\).

📉 Antécédent

Résoudre l'équation \(f(x)=k\).

⚠️ Erreurs fréquentes

❌ Confondre image et antécédent

L'image est le résultat obtenu après le calcul de f(x). L'antécédent est la valeur de x que l'on recherche.

❌ Oublier le domaine de définition

Avant tout calcul, vérifie que la fonction est bien définie pour la valeur de x utilisée.

❌ Erreurs de calcul

Remplace toujours correctement la valeur de x avant d'effectuer les opérations.

📝 Exercice

Soit la fonction :

\[ f(x)=3x-5 \]

Calculer :

\[ f(4) \]

✅ Correction

On remplace simplement \(x\) par 4 :

\[ f(4)=3\times4-5 \] \[ f(4)=12-5 \] \[ \boxed{f(4)=7} \]
